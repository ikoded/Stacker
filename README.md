# Stacker

A sleek little C++ stacking game built to explore multithreading in game logic.

## Build it
From the project root, compile the makefile:

```bash
make
```

This will generate the executable: `stacker.exe`

## Run it
Start the game by running:

```bash
./stacker.exe
```

## Clean up
To remove the build output:

```bash
make clean
```
