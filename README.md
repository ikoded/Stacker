A simple C++ game built to learn the basics of game design and multithreading. Afterward, I plan to decompile it from scratch and explore x86 assembly decompilation so I can get into security research with video games.

## Build

From the project root, compile the makefile:

```bash
make
```

This will generate the executable: `stacker.exe`

## Run it

Start the game by running:

```bash
./stacker.exe
```

## Clean up

To remove the build output:

```bash
make clean
```
